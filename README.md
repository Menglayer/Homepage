# 萌芽社区 · 官方链接与工具导航 🌱

**MengLayer（萌芽社区）** 是一个专注于 Web3 研究、策略交易与社区共建的加密社区。这里聚合了官方链接、合作交易所与券商入口，以及自研/精选的小工具，帮助成员更高效地参与市场与活动。

---

## 社区愿景 🎯
- 以数据与策略驱动的投资与研究 📊  
- 搭建高质量的工具与导航，降低行动成本 🧭  
- 通过公开透明的内容与福利，强化社区协作 🤝

---

## 功能模块一览 🧩

### 1) 小工具（Tools） 🧰
- **Falcon 空投计算器**  
  说明：用于估算活动激励与领取区间；支持弹窗内自适应展示。  

- **地址证明生成器（Address Proof Generator）**  
  说明：一键生成地址/居住证明 PDF，适用于开户/认证等场景。

- **币安Alpha助手（BN ALPHA ）**
  说明：Alpha信号助手,空投闹钟提醒。



### 2) 交易所（Exchanges） 🏦
聚合常用 CEX 合作入口：**Binance / OKX / Bybit / Gate / Bitget / Ourbit / Backpack** 等，统一卡片样式与跳转策略，提升访问效率与安全性 🔗

### 3) 券商（Broker） 📈
收录合规券商入口：**iFAST / Longbridge / Trading212** 等，便于传统市场与加密资产的多市场配置 🌐

## 交互与视觉 🌈
- **简洁信息架构**：卡片化内容、主辅信息分层、可扫描式标题与简介 🗂️  
- **多语言切换**：内置中英双语词典（i18n），标题与描述统一由词典驱动 🌍  
- **弹窗标题自动匹配**：工具弹窗标题根据点击卡片动态切换 🏷️  
- **支付链路稳定性**：识别常见收银台/支付域名（支付宝 / Stripe / 微信等），自动在新标签页打开，避免 iframe 限制导致的跳转失败 🔐  
- **移动端适配**：自适应布局、圆角卡片与柔和阴影、轻量过渡动画 📱

## 内容与文案来源 🧾
- **链接数据**：统一集中维护（工具 / 交易所 / 券商）。  
- **多语言词典**：卡片标题与描述由词典读取，便于后续批量维护。  
- **品牌素材**：按需放置于 `assets/`，支持后续替换与扩展 🖼️

## 目录概览（简要） 🗺️
```
index.html          # 页面主体（含社群弹窗、卡片容器等）
assets/
  style.css         # 样式
  app.js            # 交互逻辑（工具弹窗/支付兜底/微信群弹窗事件委托等）
  links.js          # 链接清单（工具/交易所/券商）
  i18n.js           # 多语言词典（标题/描述/按钮文案）
  images...         # 图标与素材（交易所、工具、头像、favicon 等）
```
