# 萌芽社区 · 主页 — 功能说明书

> 目标：把常用的 CEX/券商入口与一站式 Web3 工具整合在同一页面，开箱即用。

---

## 个人名片 & 互动

- 🧑‍💼 **个人资料卡**：头像、昵称、简介统一风格展示。  
- 🤝 **社交按钮**：关注 X（Twitter）、加入 Telegram/微信/Discord 群等一键直达。  
- 🧧 **微信群弹窗**：点击弹出二维码与提示文案（如“群满请加小助手…”）。

---

## 工具区 🧰

> 工具均以卡片呈现，点击后**站内弹窗**打开（个别敏感域名强制新页）。以下仅列示**核心功能点**：

- 🪂 **Falcon 空投计算器**  
  估算空投额度与价值，支持自定义参数与占比推演。

- 🎒 **Backpack 空投计算器**  
  - 支持将**积分总量**直接填入“假设看板”进行估算；  
  - 默认 `FDV = 5B`、`空投比例 = 10%` 可修改；  
  - 提供「关注我后查看结果」的交互能力（可开关）。

- 🧾 **地址证明生成器（Address Proof Generator）**  
  一键生成地址持有证明文件，适合申请/核验等使用场景。

- 🧠 **币安 Alpha 助手**  
  快速跳转使用，辅助研判与跟踪市场动向。

- 💧 **Pendle YT 计算器**  
  对应 YT 工具计算收益、期限与到期价值等关键指标。

- 🧪 **代币合约检测**  
  快速做基础合约体检，识别常见风险项与红旗。

- 🎁 **通用空投计算器**  
  估算不同规则下的空投发放与个人可得比例。

- 💸 **合约价差 / 资金费看板**  
  汇总主要交易所的资金费与价差数据，便于择机交易与对比。

- 📈 **年化复利计算器（APR/APY）**  
  - APR/APY 互转；  
  - 复利周期可选（按日/周/月/年）；  
  - 支持曲线图展示本息增长与收益轨迹。

- 🧮 **借贷风险计算器**  
  - 抵押率/LTV、清算阈值、健康度（Health Factor）等核心指标换算；  
  - 支持**循环贷（循环次数）**测算与风险边界；  
  - 预置主流借贷平台选项（如 Aave、Morpho 等，可扩展）；  
  - 可引入特定资产（如 PT sUSDe）价格参与计算。

- 🧊 **Uniswap v3（CLAMM）区间做市 & IL 模拟器**  
  - 自定义 LP 区间、仓位规模与价格路径；  
  - 计算无常损益（IL）与区间内收益表现；  
  - 内置图表展示，支持自适应与边界保护。

- 📊 **Kaito 看板 / Kaito Yaps**  
  通过外部看板聚合热点与情绪，辅助研究判断。

> 注：工具卡片均配有图标与中英标题/描述，便于快速识别。
