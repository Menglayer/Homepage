## ④ 工具区 🧰（点击卡片打开）

> 实用操作为主，适合研究、估算、对比和做决策。

- 🪂 **Falcon 空投计算器**  
  - 选择或输入你的关键参数（如地址得分/任务完成度等）。  
  - 一键计算预估**可得空投与价值区间**，并显示对整体池子的**占比**。  
  - 适合活动期快速心里有数，便于制定“是否继续做任务/拉满某项”的取舍。

- 🎒 **Backpack 空投计算器**  
  - 在“假设看板”填入**积分总量**与**个人积分**。  
  - 可调 `FDV`（默认 **5B**）与 `空投比例`（默认 **10%**），系统实时给出：  
    - **1 分的预测价值**  
    - **我的空投价值**与**我的占比**  
  - 若有开启「关注我后查看结果」：按提示跳转关注，回来后自动显示计算结果。

- 🧾 **地址证明生成器（Address Proof）**  
  - 点击后在**新标签页**打开。  
  - 按页面提示输入地址并完成签名，即可生成**地址持有证明**文件，用于社群认证、申领或对接方核验。

- 📈 **年化复利计算器（APR/APY）**  
  - 输入本金、年化、时长与**复利周期**（日/周/月/年）。  
  - 图表展示**本金增长曲线**；支持 **APR ↔ APY** 互算。  
  - 适合：比较不同平台/产品在**不同复利频率**下的真实收益差异。

- 🧮 **借贷风险计算器**  
  - 选择平台（如 Aave、Morpho）与抵押资产，填写抵押额/LTV/清算阈值等。  
  - 输出：**可借上限**、**健康度（HF）**、**触发清算的价格跌幅**。  
  - **循环贷**：可模拟**循环次数**对头寸与风险的影响，找到“收益-风险”的平衡位。  
  - 支持引入特定资产（如 PT sUSDe）价格进行试算。

- 🧊 **Uniswap v3（CLAMM）区间做市 & IL 模拟器**  
  - 输入当前价格、设置**做市价格区间**与注入金额（支持自定义 LP）。  
  - 输出：区间内的**费用收益**与**无常损益（IL）**，并可观察不同价格路径的影响。  
  - 用途：上 LP 前**评估区间与头寸风险**，避免“价格一出区间就闲置”。

- 🧪 **代币合约检测**  
  - 粘贴合约地址即可进行基础体检，快速查看常见**风险项/红旗**（如可增发、可拉黑、代理合约等）。  
  - 适合：初筛，非审计替代。

- 🎁 **通用空投计算器**  
  - 选择派发规则（积分/等级/权重等），输入你的参数，估算**可得份额与价值**。  
  - 适合不同项目的**通用收益试算**。

- 💸 **合约价差 / 资金费看板**  
  - 汇总多家交易所的资金费率、现货-永续**价差**、多空偏置。  
  - 支持并排**对比**，便于择机与跨所策略。

- 📊 **Kaito 看板 / Kaito Yaps**  
  - 外部数据看板，聚合热点、话题与情绪。  
  - 打开后可筛选主题、研究热度走势，辅助选题和交易方向。

- 🧠 **币安 Alpha 助手**  
  - 直达官方 Alpha 页面，用于跟踪策略与市场洞察。

> 所有工具卡片都提供**图标 + 中英标题/说明**，一眼识别；不支持内嵌的会自动走新标签页。
